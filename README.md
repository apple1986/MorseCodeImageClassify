# Morse Code Image Classification

This repository provides a **Morse code image dataset** and a demo for image classification using a noise-adaptive network.

---

## Project Overview

Morse code image classification is a challenging task due to the noise and variability in the dataset. This project implements the **Noise Adaption Network (NAN)** to improve robustness and accuracy for Morse code image classification. The provided demo allows users to train and test the model on our curated dataset.

---

## Dataset

The Morse code image dataset can be downloaded here:  
[Google Drive Link](https://drive.google.com/file/d/1mKh23Yk4ygNkye3fanqPapx8X9Qs1z9b/view?usp=sharing)

**Instructions:**

1. Download the dataset and unzip the file.  
2. Run the demo script:

```bash
python train_test_morse_code.py
```

This will train the model on the dataset and evaluate its performance.

---

## Demo

The provided script `train_test_morse_code.py` demonstrates:

- Loading the Morse code dataset  
- Training the Noise Adaption Network  
- Evaluating the model on test images
